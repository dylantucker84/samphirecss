# Samphire

Samphire is the CSS system built for modern websites that prioritises
pragmatism and simplicity.

------------------------------------------------------------------------

## Why Samphire?

Not all CSS frameworks are built for websites. Many are optimised for complex apps and developer-heavy workflows - powerful, but bloated and pipeline-dependent. SamphireCSS takes a minimal approach for easy to maintain, content-first websites.

-   Minimal surface area\
-   No required build step\
-   Utility-driven layout system\
-   Clear responsive breakpoints\
-   Unopinionated design tokens

------------------------------------------------------------------------

## Installation

### CDN

``` html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dylantucker84//samphire@v1.0.0/dist/samphire.min.css">
<script src="https://cdn.jsdelivr.net/gh/dylantucker84//samphire@v1.0.0/dist/samphire.min.js"></script>
```

### Download

Download the latest release from GitHub and include:

    dist/samphire.min.css
    dist/samphire.min.js

## Breakpoints

-   sm ≥ 576px\
-   md ≥ 768px\
-   lg ≥ 1024px\
-   xl ≥ 1280px\
-   xxl ≥ 1440px

## Philosophy

Samphire prioritises:

-   Explicit utilities over implicit behaviour\
-   Predictable layout control\
-   Minimal abstraction\
-   Website-first architecture
